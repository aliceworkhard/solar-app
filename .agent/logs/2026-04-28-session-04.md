## Implementation Notes

- Kept BLE native, protocol command HEX, and `deviceController.ts` semantics unchanged.
- `readStatus()` remains non-blocking at the command layer and does not wait for BLE notify.
- Added App-level status polling with `STATUS_POLL_INTERVAL_MS = 5000`.
- Polling only runs when `DeviceStatus.connected === true`, `connectionState === "ready"`, and no poll is already in flight.
- Swipe disconnect uses a left-swipe threshold and only exposes the action for the currently connected card.
- UI still does not parse raw HEX directly; status values come from `DeviceStatus`.
