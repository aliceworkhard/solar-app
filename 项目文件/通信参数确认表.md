## 5. 结论口径（当前）

- 当前使用参数：
  - `serviceUUID = FFF0`
  - `writeUUID = FFF1`
  - `notifyUUID = FFF2`
- 当前写入方式：固定 `write`。
- 调试页 RAW 发送：只发送，不等待 BLE 回包。
- 业务命令默认：只发送，不等待 BLE 回包；notify 若有数据则被动解析。
- 当前协议帧格式：`FF CE + LEN + DESC + CMD + SUB + DATA + 30 + SUM`。
- 当前 5 条 RF 控制命令：开/关、增加亮度、降低亮度、读参数、读状态。
- 当前未决项：MTU 真机最终值、20 次性能采样结果、是否存在设备侧隐藏握手要求、开/关是否为单一切换命令。
