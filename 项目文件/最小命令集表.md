# 最小命令集表（第一阶段）

更新时间：2026-04-22

## 1. 使用说明

- 目标：先固化 5 条可验证闭环命令，支撑“发得出 + 收得到 + 解析正确”。
- 约束：所有命令必须能在调试页复现，且有回包判定规则。
- 说明：`HEX` 以空格分隔（示例：`AA 55 01 00 56`）。

## 4. 收包解析规则（先按此实现）

1. 先做缓冲区拼包，不能假设一次 notify 就是一帧。
2. 先按“帧头 + 长度 + 校验”进行切帧。
3. 不完整帧留在缓冲区，等待下一包拼接。
4. 校验失败帧记录日志并丢弃，不更新 UI。
5. 每条命令必须有“请求-回包”关联日志。
