# Project Policy

## Priority

- `P0`：不做会阻塞 BLE / 协议 / 真机闭环。
- `P1`：影响主要体验或验收，但不阻塞基础链路。
- `P2`：整理、增强、优化、归档。

## Status

- `Proposed`：已提出，等待用户确认。
- `Approved`：用户已确认，可以实施。
- `In Progress`：正在实施。
- `Blocked`：缺资料、缺设备或环境阻塞。
- `Done`：已完成并验证或记录无法验证原因。

## Approval Record

Approval records live in `.agent/approvals/`.

Minimum content:

- Plan file.
- User approval wording.
- Approved scope.
- Files allowed to modify.
- Verification expected.
